### 功能

 1.用户：包括登录、注册、个人信息管理、群组信息管理<br>
 2.消息：实现了群聊、私聊。包括表情图片文件发送，其中表情可以快捷输入，图片可以使用ctrl+v粘贴发送<br>
 3.群组：可以修改群组信息，还可以通过链接加入群组<br>
 4.favico提醒：消息提醒<br>
